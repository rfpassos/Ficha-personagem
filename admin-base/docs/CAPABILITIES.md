## 📋 Rules (Regras Globais)

Definidas no `GEMINI.md`, aplicam-se a TODAS as interações:

1. **Intelligent Routing** - Seleção automática de agents
2. **Socratic Gate** - Perguntas antes de implementar features complexas
3. **Clean Code** - Código pragmático, sem over-engineering
4. **Speaking Portuguese** - Comunicação em PT-BR
5. **File Dependency Awareness** - Verifica dependências antes de editar
6. **Final Checklist Protocol** - Validação obrigatória antes de finalizar
