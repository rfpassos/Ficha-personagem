# frontend-moderno

Skill para criar interfaces modernas, amigáveis e visualmente atraentes usando Next.js + React + Tailwind v4 + shadcn/ui + Framer Motion.

## Objetivo

Evitar o visual "padrão de mercado" — criar interfaces que pareçam crafted à mão, não geradas por IA.

## Uso

Ative esta skill quando:
- Criar novos componentes UI do zero
- Projetar interfaces interativas com animações suaves
- Construir ou estender um design system
- Personalizar componentes shadcn/ui além do padrão
- Adicionar polish visual em funcionalidades existentes

## Estrutura

```
frontend-moderno/
├── SKILL.md          # Instruções principais
├── README.md         # Este arquivo
└── templates/        # Exemplos de componentes
    ├── button.tsx
    ├── card.tsx

    └── animations.ts
```

## Princípios

1. **Personalidade** — paletas não convencionais, micro-interações, tipografia intentional
2. **Tailwind v4** — usar CSS variables, não valores hardcoded
3. **Animações** — Framer Motion com padrões consistentes
4. **Acessibilidade** — teclado, ARIA, contraste WCAG AA
